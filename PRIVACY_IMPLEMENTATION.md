# Privacy implementation

Planner preferences are stored under `worldtime-grid-planner-v1` in localStorage. They include selected cities, order, work settings, display format, date and duration. The reset control deletes that value.

Share URLs include only zones, date, selected UTC instant, duration and time format. Meeting titles, names, email addresses and conferencing links are not automatically included. ICS files are generated locally and user text is escaped.

No account or application database exists. Advertising is disabled by default. If AdSense is enabled, the operator is responsible for an appropriate consent configuration and for keeping the public privacy and cookie text aligned with actual analytics and advertising products.
