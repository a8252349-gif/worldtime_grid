# 전체 도시 삭제 기능 수정

## 변경 사항

- 전체 플래너의 도시 검색창 오른쪽에 언어별 `모든 도시 삭제` 버튼을 추가했습니다.
- 클릭하면 현재 선택된 도시를 한 번에 모두 제거합니다.
- 모든 도시를 지운 상태도 localStorage에 저장되므로 새로고침 후 자동으로 예시 도시가 다시 생기지 않습니다.
- 공유 URL로 열린 플래너에서 전체 삭제를 누르면 기존 `zones`, `time` 쿼리가 다시 도시를 복원하지 않도록 현재 경로에서 쿼리스트링을 제거합니다.
- 도시가 없는 상태에는 `예시 도시 3개 불러오기` 버튼을 표시합니다.
- 기존 하단의 초기화 기능은 `기본 설정으로 되돌리기`로 이름을 바꿔 역할을 구분했습니다.
- 도시가 없을 때 저장 조합, ICS, 공유 및 복사 버튼을 비활성화합니다.
- 모바일에서는 전체 삭제 버튼이 검색창 아래에서 가로 폭 전체를 사용합니다.

## 언어별 버튼

- English: Clear all cities
- 한국어: 도시 전체 삭제
- 日本語: すべての都市を削除
- Español: Eliminar todas las ciudades
